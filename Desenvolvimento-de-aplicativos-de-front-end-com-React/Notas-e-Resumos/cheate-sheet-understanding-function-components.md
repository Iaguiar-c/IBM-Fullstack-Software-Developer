Cheat Sheet: Understanding Function Components with Array and DOM Manipulation

| Function Component with Array and DOM | Description | Code Example |
|---|---|---|
| Function Component with function keyword | Function component starts with function keyword along with name of the component and includes html tags within return. It also exports component name by default | `import React from 'react'`<br>`function Extra() {`<br>`  return (`<br>`    <>`<br>`    <p>This is paragraph</p>`<br>`    </>`<br>`  )`<br>`}`<br>`export default Extra` |
| Function Component with arrow function | Function component starts with variable type along with name of the component and includes html tags within return. It also exports component name by default | `import React from 'react'`<br>`const Extra = () => {`<br>`  return (`<br>`    <div>Extra</div>`<br>`  )`<br>`}`<br>`export default Extra` |
| Props in function component | Props can be sent from parent component as attribute along with child component | `import React from 'react'`<br>`import ChildComponent from './ChildComponent'`<br>`function ParentComponent () {`<br>`    let title='Project Manager';`<br>`    return (`<br>`    <>`<br>`    <ChildComponent title={title}/>`<br>`    </>`<br>`  )`<br>`}`<br>`export default ParentComponent` |
| Access Props within child function component | Props can be accessed easily within the child function component using props.variable_name | `import React from 'react'`<br>`const ChildComponent = (props) => {`<br>`  return (`<br>`    <>`<br>`    <p>The title is {props.title}</p>`<br>`    </>`<br>`  )`<br>`}`<br>`export default ChildComponent` |
| Event handling in class component | Events such as click event can be performed by calling function which is declared before return of function component | `import React from 'react'`<br>`const Extra = (props) => {`<br>`    function show(){`<br>`        console.log('Show function');`<br>`    }`<br>`  return (`<br>`    <>`<br>`    <p>The title is {props.title}</p>`<br>`    <button onClick={()=>show()}>Click Here</button>`<br>`    </>`<br>`  )`<br>`}`<br>`export default Extra` |
| State management in function component | State management can be done easily with useState() hook | `import React, { useState } from 'react'`<br>`const StateManagement = () => {`<br>`    const[name,setName]=useState('John');`<br>`  return (`<br>`   <>`<br>`   <h1>State Management using useState</h1>`<br>`    <p>The name is {name}</p>`<br>`   </>`<br>`  )`<br>`}`<br>`export default StateManagement` |
| Array Declaration | Array can be declared in square brackets | `const names = ['Alice', 'Bob', 'Charlie'];` |
| Stateful Array | Array can be declared using useState | `const [todos, setTodos] = useState(['Learn React', 'Build Project']);` |
| Dynamically Constructed Arrays | Arrays can be constructed dynamically based on application logic or received data | `const numbers = [];`<br>`for (let i = 0; i < 10; i++) {`<br>`  numbers.push(i);`<br>`}` |
| Array map() method | The map() method is commonly used to iterate over each element of an array and return a new array of React elements | `const items = ['Apple', 'Banana', 'Orange'];`<br>`const itemList = items.map((item, index) => <li key={index}>{item}</li>);`<br>`return <ul>{itemList}</ul>;` |
| for...of Loop | You can use the for...of loop to iterate over the elements of an array: | `const items = ['Apple', 'Banana', 'Orange'];`<br>`for (const item of items) {`<br>`  console.log(item);`<br>`}` |
| Rendering a List of Items | You can render a list of items by mapping over an array and returning a JSX element for each item | `import React from 'react';`<br>`function ArrayComponent() {`<br>`  const items = ['Autumn', 'Spring', 'Summer','Winter'];`<br>`  return (`<br>`    <div>`<br>`      <h1> Seasons Names</h1>`<br>`      <ul>`<br>`        {items.map((item, index) => (`<br>`          <li key={index}>{item}</li>`<br>`        ))}`<br>`  </ul>`<br>`    </div>`<br>`  );`<br>`}`<br>`export default ArrayComponent;` |
| Adding and removing items in array | You can add or remove items from an array using state and React's setState method | `import React, { useState } from 'react';`<br>`function MyComponent() {`<br>`  const [items, setItems] = useState(['Autumn', 'Spring', 'Winter','Summer']);`<br>`  const [inputValue, setInputValue] = useState('');`<br>`  const addItem = () => {`<br>`    setItems([...items, inputValue]);`<br>`    setInputValue('');`<br>`  };`<br>`  const removeItem = (index) => {`<br>`    const newItems = [...items];`<br>`    newItems.splice(index, 1);`<br>`    setItems(newItems);`<br>`  };`<br>`  return (`<br>`    <div>`<br>`      <h1>Fruits</h1>`<br>`      <ul>`<br>`        {items.map((item, index) => (`<br>`          <li key={index}>`<br>`            {item}`<br>`            <button onClick={() => removeItem(index)}>Remove</button>`<br>`          </li>`<br>`        ))}`<br>`      </ul>`<br>`      <input`<br>`        type="text"`<br>`        value={inputValue}`<br>`        onChange={(e) => setInputValue(e.target.value)}`<br>`      />`<br>`      <button onClick={addItem}>Add</button>`<br>`    </div>`<br>`  );`<br>`}` |
| Conditional rendering using ternary operator | You can conditionally render components based on the content of an array | `import React, { useState } from 'react';`<br>`function ArrayComponent() {`<br>`    const [items, setItems] = useState(['React JS','Vue JS','Angular JS','Vanilla JS']);`<br>`  return (`<br>`    <div>`<br>`      <h1>Front End Languages</h1>`<br>`      {items.length > 0 ? (`<br>`        <ul>`<br>`          {items.map((item, index) => (`<br>`            <li key={index}>{item}</li>`<br>`          ))}`<br>`        </ul>`<br>`      ) : (`<br>`        <p>No Front End language is available</p>`<br>`      )}`<br>`    </div>`<br>`  );`<br>`}`<br>`export default ArrayComponent;` |
| inline style in react | Inline style can be applied within the tag as an attribute within double curly braces | `import React from 'react';`<br>`function MyComponent() {`<br>`  return (`<br>`    <div style={{ backgroundColor: 'lightblue', padding: '20px', borderRadius: '5px' }}>`<br>`      <p style={{ color: 'white', fontSize: '18px' }}>This is a paragraph with inline styles.</p>`<br>`    </div>`<br>`  );`<br>`}`<br>`export default MyComponent;` |
| Style using object | Style can be applied as an object like inline style | `import React, { useState } from 'react';`<br>`function ToggleMessage() {`<br>`  const [isVisible, setIsVisible] = useState(true);`<br>`  const toggleVisibility = () => {`<br>`    setIsVisible(!isVisible);`<br>`  };`<br>`  const messageStyle = {`<br>`    display: isVisible ? 'block' : 'none',`<br>`    color: 'green',`<br>`    fontSize: '18px',`<br>`    marginTop: '10px'`<br>`  };`<br>`  return (`<br>`    <div>`<br>`      <h2>Toggle Message</h2>`<br>`      <button onClick={toggleVisibility}>`<br>`        {isVisible ? 'Hide Message' : 'Show Message'}`<br>`      </button>`<br>`      <p style={messageStyle}>This is a hidden message.</p>`<br>`    </div>`<br>`  );`<br>`}` |
